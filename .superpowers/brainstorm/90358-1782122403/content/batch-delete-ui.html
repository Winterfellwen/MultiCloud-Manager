批量删除交互方式

对话列表中如何选择要删除的对话？

A

复选框模式

每个对话左侧显示复选框，顶部有"全选"按钮和"删除选中"按钮。点击进入多选模式。

全选 (3/10) 🗑 删除选中

B

右键/长按菜单

右键点击对话弹出菜单，包含"删除"、"删除多个"等选项。选择"删除多个"后进入多选模式。

如何扩容 ECS 实例

📋 复制链接
🗑 删除
☑️ 删除多个...
C

编辑模式切换

顶部添加"编辑"按钮，点击后列表进入编辑模式，每行出现复选框。类似 iOS 编辑模式。

✓ 完成 已选 2 项 🗑 删除
