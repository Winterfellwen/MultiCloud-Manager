关键区别 vs 旧卡片设计

✅ 新设计

  • 工具=文本行，没有盒子
  • 缩进 + 分隔线区分
  • 展开细节用浅灰背景块
  • 复制按钮在展开后显示
  • 所有样式节省 ~70% CSS

❌ 旧卡片

  • 每个工具一个卡片盒子
  • 彩色左条 + 边框
  • 进度条 + 时间戳
  • 分离于 AI 消息之外
  • ~118 行 CSS，600+ JS
